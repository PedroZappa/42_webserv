# Command Glossary

## NGINX
 
```sh
# Stop Webserver
sudo systemctl stop nginx
# Start Webserver
sudo systemctl start nginx
# Restart Webserver
sudo systemctl restart nginx
# Reload Webserver
sudo systemctl reload nginx
# Disable Webserver
sudo systemctl disable nginx
# Enable Webserver
sudo systemctl enable nginx
# Test Webserver
sudo nginx -t
# Invoke NginX executable

nginx -s signal
```
